# Calculadora web en la nube

Aplicación académica sencilla desarrollada con HTML, CSS y JavaScript para demostrar el proceso de despliegue desde GitHub hacia Render.

## Funciones

- Suma, resta, multiplicación y división.
- Operación de porcentaje.
- Botón para borrar el último carácter.
- Botón para reiniciar la calculadora.
- Uso mediante botones o teclado.
- Diseño adaptable a computadoras y teléfonos.

## Archivos del proyecto

- `index.html`: estructura principal de la aplicación.
- `styles.css`: diseño visual y adaptación a distintas pantallas.
- `script.js`: funcionamiento de la calculadora.
- `render.yaml`: configuración opcional para Render.

## Ejecución local

Abra el archivo `index.html` en un navegador web.

## Despliegue en Render

1. Suba todos los archivos a un repositorio de GitHub.
2. Ingrese a Render y conecte su cuenta de GitHub.
3. Seleccione **New > Static Site**.
4. Conecte el repositorio.
5. Use la rama `main`.
6. Deje vacío el campo **Build Command**.
7. Escriba `.` en **Publish Directory**.
8. Presione **Create Static Site**.
9. Espere a que el despliegue muestre el estado **Live**.
10. Copie la URL pública terminada en `.onrender.com`.

## Evidencias sugeridas

- Repositorio de GitHub con los archivos visibles.
- Ventana de autorización o repositorio conectado en Render.
- Configuración del sitio estático.
- Estado `Live` y URL pública.
- Aplicación abierta en el navegador y una operación resuelta.
